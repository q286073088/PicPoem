# PicPoem

PicPoem 是一个用图片展示古诗词的学习资源项目。项目将小学、初中、高中阶段的古诗词按年级和册别整理成图片，通过可视化场景、意象和情绪，帮助孩子更快理解诗词内容，提高背诵、复习和课堂预习效率。

## 项目特点

- 按年级和上下册分类，便于跟随教材进度查找。
- 每首诗词对应一张或多张图片，适合预习、复习、背诵和亲子共读。
- 通过画面辅助理解诗词中的人物、时间、地点、情绪和关键意象。
- 资源以图片文件为主，打开 GitHub 文件夹即可浏览和下载。

## 使用方式

1. 根据孩子所在年级进入对应文件夹。
2. 打开诗词图片，先看画面理解诗词场景和情绪。
3. 再对照课文原文进行朗读、背诵和重点字词复习。
4. 复习时可以先遮住文字，只看图片回忆诗句和作者。

## 文件命名建议

建议继续使用统一命名格式：

```text
诗名-朝代-作者[-课文信息或序号].png
```

示例：

```text
枫桥夜泊-唐-张继-第21课-古诗词三首-1.png
望岳-唐-杜甫-2.png
```

如果一首诗词有多张图片，可以在文件名末尾追加 `-1`、`-2` 等序号。

## 维护建议

- 新增图片前尽量先压缩，避免仓库体积快速膨胀。
- 单张图片建议控制在几 MB 以内，尽量不要上传超大 PNG。
- 若后续扩展到高中或加入更多版本教材，可继续按 `年级 + 上/下册` 新建目录。
- 若图片资源来源于第三方素材、字体、插画或近现代作品，请确认授权后再公开发布。

## 项目愿景

让古诗词不只停留在文字背诵上，而是通过画面建立记忆线索，帮助孩子把诗词中的景、情、意联系起来，形成更自然、更持久的理解和记忆。
